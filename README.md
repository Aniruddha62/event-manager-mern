# Event Manager (MERN Stack)

This is a full-stack Event Management application built using the MERN stack.

It allows users to create, update, and manage events with secure authentication.

## 🚀 Tech Stack

- React
- Node.js
- Express
- MongoDB
- JWT Authentication

## ✨ Features

- Create events
- Update events
- Delete events
- User authentication
- Responsive UI
